# Currency Rates
A fetching currency rates tool and display rates on your terminal.

## Install
```
$ npm i -g currency-rates
$ currency-rates
```

## Customize
```
$ # -r is rows and -c is columns
$ currency-rates -r USD,CAD,EUR -c GBP,AUD,JPY
```

## Inverse
```
$ currency-rates -i
```

## Show Available Countries
```
$ currency-rates -l
```
